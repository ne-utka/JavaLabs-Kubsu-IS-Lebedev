1) main() — точка входа программы, с нее начинается выполнение.
2) String[] args — массив строк с аргументами командной строки.
3) JDK — набор для разработки (компилятор, инструменты), JRE — только среда выполнения.
4) Свойство length.
5) println печатает и переводит строку, print — без перевода.
6) Через оператор + или StringBuilder.
7) void — метод ничего не возвращает.
8) static — принадлежит классу, а не объекту.
