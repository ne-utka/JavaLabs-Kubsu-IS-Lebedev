1) Сигнатура метода — это его имя и список параметров с типами и порядком.
2) Конструктор нужен для создания объекта и инициализации его полей.
3) Класс — описание структуры и поведения, объект — конкретный экземпляр класса.
4) Пакеты группируют классы, помогают организовать код и избегать конфликтов имен.
5) new выделяет память и вызывает конструктор для создания объекта.
6) Пользовательский конструктор имеет параметры и задается в классе; конструктор по умолчанию без параметров, создается автоматически, если нет других.
7) Операторы import подключают классы из пакетов, чтобы писать их без полного имени.
